# Web-boilerplate 🌍⚒

Featuring:

* Webpack:
    * Webpack dev-server
    * Webpack mocking
    * Webpack proxy
    * Optimized build
* React
* React-router
* React-bootstrap
* Docker:
    * Dockerfile
    * docker-compose config
    * Nginx config 
* Scripts:
